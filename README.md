# GigaChat Telegram Bot

Telegram бот на базе GigaChat от Сбера.

## Возможности

- 🤖 Ответы на любые вопросы
- 💬 Диалоговый режим
- 🎨 Креативные тексты и идеи
- 📊 Система лимитов
- 👑 Админ панель

## Установка

```bash
# 1. Клонируй репозиторий
git clone https://github.com/Olga768/telegram-gigachat-bot.git
cd telegram-gigachat-bot

# 2. Установи зависимости
pip install -r requirements.txt

# 3. Настрой .env файл
# TELEGRAM_BOT_TOKEN=твой_токен
# GIGACHAT_AUTH_KEY=твой_ключ

# 4. Запусти бота
python bot.py
